How to control my Homepage settings?

1- Please check "WordPress Dashboard > Appearance > Customize".

2- Click on "Static Front page".

3- Either choose "Your latest posts".

4- Or, choose "A static page" in case you want to set a custom page as your homepage.

Check this part of the theme documentaion to learn more about How to Set your Homepage

My website turned into a white screen! what's going on ?!

1- Please make sure that all the theme files are uploaded correctly, you can re-upload them again to make sure.

2- Try to increase the memory limit allocated by WordPress, check this link for help.

3- Please purge your website cache if you are using any caching plugin, also consider deleteing CDN cache in case you are using one, another thing to clear is your browser cache.

Slider and Homepage builder are merged into our new "Page Builder" which can be found in the default WordPress page eidtor, just go to (WordPress Dashboard > Pages > Add New) and you will find the "Page Builder" button as shown in this screenshot:

How to downgrade to Sahifa 4.x.x ?

You will need to upload the old version of the theme files to your website, then navigate to (WordPress Dashboard > Appearance > Customize) and select "Your latest posts" under "Static Front Page" just like in this screenshot:

Where is the "Social Counter" widget ?

Social Counter widget is replaced with a whole new pre-installed plugin "Arqam Lite", please find it in the Widgets sections under the name "Arqam Lite - Social Counter"

My website is showing some English Strings although I'm using a translated version of the theme?!

The translation files have been changed in the new version, you can now easily translate the theme fron-end strings throw (Sahifa TiePanel > Translation) or directly translate the languages files, to do so check this guide.
