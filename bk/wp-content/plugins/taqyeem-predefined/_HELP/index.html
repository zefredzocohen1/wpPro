Installing The Plugin

Uploading The Plugin via FTP to Wordpress

You can also install The Plugin via FTP

  1. Decompress the .zip file you downloaded from CodeCanyon.
  2. Find the taqyeem-predefined folder (this directory is created when you unzip the file).
  3. Upload the taqyeem-predefined folder to your wp-content/plugins directory.
  4. Navigate to your Control Panel: Plugins
  5. Under Taqyeem Predefined Criteria, click Activate

Installing The Plugin from the WordPress Dashboard

You can install The Plugin via the WordPress plugin uploader without unzipping the file.

  1. Log into your WordPress admin panel
  2. Navigate to Plugins > Add New
  3. Click Upload
  4. Click Choose File and select the taqyeem-predefined download zip.
  5. Click Install Now.

Adding a Predefined Criteria Set

  1. Navigate to Posts > Predefined Criteria tab
  2. Click the "Add New Review Criteria Set" button to create a new set.
  3. In the pop-up window Enter a name for the set then hit Ok button .
  4. Enter a name for the Review Criteria .
  5. Click on Add Another Review Criteria button to add more review criteria.
  6. Once its all done, make sure you click the Save button.

Using a Predefined Criteria Set

  1. Check this for how to add a review post.
